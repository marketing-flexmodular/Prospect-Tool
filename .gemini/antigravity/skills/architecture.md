# Prospect Tool Project Architecture Skill
This skill ensures development follows the unified module architecture.

## Module Structure
- **Explore**: Google Maps lead retrieval and enrichment.
- **Lists**: Saved groups of prospects.
- **Pipeline**: CRM Kanban board for deal tracking.
- **Analytics**: Performance and usage metrics.

## Key Services
- **Gemini Service**: Handles AI-driven lead enrichment and Maps search.
- **Location Service**: Provides Brazilian city data for autocomplete.
- **Auth Context**: Manages session and usage quotas (currently local).

## Tech Stack
- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS (Santiz Crimson theme)
- **UI Components**: Shadcn UI (planned)
- **Icons**: Lucide React
- **Backend**: Supabase (planned integration)
